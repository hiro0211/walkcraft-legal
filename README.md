# walkcraft — Legal pages

Bilingual (日本語 / English) Privacy Policy and Terms of Service for the **walkcraft** iOS app,
hosted via GitHub Pages.

- `index.html` — hub / landing
- `privacy.html` — Privacy Policy / プライバシーポリシー
- `terms.html` — Terms of Service / 利用規約
- `assets/style.css` — shared styling (mirrors the in-app dark/amber theme)
- `assets/lang.js` — language toggle (default Japanese, persisted in `localStorage`)

Language preference is remembered in the browser. Static, self-contained, no build step.
